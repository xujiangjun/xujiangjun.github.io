Spring简介

Spring 对 Java 版本的要求

Spring 4.0 基于 Java 6.0，全面支持 Java 8.0。运行 Spring 4.0 必须使用 Java 6.0 以上版本。

Spring 4.0 新特性

  1. DateTimeFormat注解可以通过配置格式化”yyyy-MM-dd”格式的日期字符串。
  2. 空指针终结者Optional<>、依赖注入和SpringMVC参数接收可选化。
  3. 核心容器增强：支持泛型依赖注入、对Cglib类代理不再要求必须有空参构造器。
  4. 从 Spring 4.0 开始，SpringMVC 基于 Servlet 3.0 开发。
  5. 使用 WebSocket 需要考虑浏览器版本（IE<10 不支持），目前主流的浏览器都能很好地支持WebSocket。

Spring 子项目

Spring Boot、Spring Cloud、Spring Data、Spring Security、Spring Session

如何获取 Spring

目前，Spring 不再提供直接下载方式，只能使用Maven或Gradle构建来下载Spring的构建包。
