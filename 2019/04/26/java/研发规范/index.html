研发规范

  1. 提供的dubbo服务应尽量保证接口的幂等性（主要是新增和修改接口），因为dubbo调用存在重试机制。

  2. MQ消费者接收消息处理也需要保证幂等性，避免重复消费引起异常。

  3. 创建表默认需要包含id、gmt_create和gmt_modified三个字段，修改时间默认设置为创建时间。
